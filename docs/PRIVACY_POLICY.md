# Pokopia Builder Privacy Policy

Last updated: April 26, 2026

Pokopia Builder is an offline-first macOS planning app. The app developer does not collect, sell, or share personal data from the app.

## Data Collection

Pokopia Builder does not collect analytics, advertising identifiers, diagnostics, contact information, location data, or usage data for the developer.

## Optional Prompt Generation

Pokopia Builder includes optional prompt-based build generation. The built-in local mode runs on the device. If a user chooses to configure OpenAI or a local Ollama server, prompts are sent only to the service the user configured for that feature. API keys and settings are stored locally on the user's Mac.

## Local Files

The app may read local catalog assets and optional user-provided 3D model files selected by the user. These files remain on the user's Mac and are not uploaded by Pokopia Builder.
